<h1 align="center">BountyBot Legacy</h1>

Welcome to the Legacy branch of BountyBot, a resurrection of BountyBot beta v0.9.3.

I've made a branch for this version, because while it will not be updated with new features or minor bug fixes, I am expecting to have to fix some major bugs in order to get it working.

This is very legacy code, with a pretty poor architecture. BountyBot release v1.0 was planned to be a much more maintainable rewrite, with base level sanity features like support for an actual database instead of JSON flat files, but sadly I don't have time to make it.

Since this branch is not 'maintained', please do not submit issues here. If you have a bug report or feature request, please report it in discord.
